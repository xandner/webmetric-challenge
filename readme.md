## Pull and Run:
- Pull project with command:
```bash
 git clone git@github.com:xandner/webmetric-challenge.git
 ```

 - run this command without any change:
 ```bash
cp .env.example .env
 ```
 ` ! its dont needs to change any thing in .env file`

 - run project with command 
 ```bash
docker compose up
```
